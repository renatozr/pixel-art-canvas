## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/renatozr/pixel-art-canvas.git
   ```
2. Enter the directory
   ```sh
   cd pixel-art-canvas
   ```
3. Run the project with Live Server or open the `index.html` file with your browser
